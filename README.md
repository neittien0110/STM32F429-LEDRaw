# EMBEDED SYSTEM / HỆ NHÚNG: Kit STM32F429 - Hello world với đèn Led

## Mục tiêu

Lập trình để tăt/bật đèn Led có sẵn trên board, bằng cách tạo Project mới theo hình thức chọn __MCU STM32F429zIT6 Board__ 

> Lưu ý: nếu tạo Project mới theo hình thức chọn Board STM32F429-DISK1 thì xem project <https://github.com/neittien0110/LEDBoilerPlate> 

## Cấu hình

Module|Tên Pin trên MCU|Tên Pin gợi nhớ trên board
--|--|--
Green LED trên board|GPIO_PIN_13, GPIOG = PG13|LD3
Red LED trên board|GPIO_PIN_14, GPIOG = PG14|LD4
GND|GND|GND
